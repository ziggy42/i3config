# i3config
My personal configuration for i3.

## Hardware
`ThinkPad X1 Carbon`

## Distro
`Fedora 27`

## Dependencies
```bash
sudo dnf install i3 i3status dmenu i3lock xbacklight feh network-manager-applet lxappearance
```

## Installation
```bash
link i3.config ~/.config/i3/config
link i3status.config ~/.config/i3status/config
```
